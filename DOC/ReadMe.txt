功能描述
================================================================================
本样例主要涉及通过串口与A39C-T400A22S1a进行通信，实现LoRa传输及控制


引脚设置
================================================================================
PA02：LoRa_RXD
PA03：LoRa_TXD
PA04: LoRa_MOD1
PA05: LoRa_MOD0


学习目的
================================================================================
单片机串口2与A39C-T400A22S1a通信，了解LoRa通信的相关操作步骤

串口调试助手
================================================================================
使用步骤
1、MCU的PA02、PA03、PA04、PA05引脚与LoRa模块的RXD、TXD、MOD1、MOD0引脚正确连接，LoRa模块的供电为5.0V
2、重新编译并运行
3、在调试窗口观察打印信息，如果与实际一致，运行正常
